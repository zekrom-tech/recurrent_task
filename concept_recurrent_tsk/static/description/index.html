Setup Instructions

Step 1: Enable Developer Mode

Go to Settings → General Settings, scroll down and click "Activate the developer mode" at the bottom of the page.

Step 2: Enable Recurring Tasks

In Settings, search for "recurring". Under Project → Tasks Management, enable the "Recurring Tasks" option and click Save.

How It Works

1. Enable Recurring Task

Check the "Recurring Task" option on any project task to turn it into a template.

2. Configure Recurrence

Set interval (e.g., every 2 weeks), start date, weekdays, and stop condition.

3. Preview & Save

Review upcoming occurrences instantly with the live preview, then save.

4. Automatic Execution

A scheduled cron job creates new tasks automatically on the specified dates.

Step-by-Step Guide: Creating a Recurrent Task

1
Navigate to Project Tasks

Go to Project → Tasks menu and either create a new task or open an existing one.

2
Enable Recurring Task

In the task form, click the "Recurring Task" icon/checkbox (usually located at the top-right). This will reveal the "Recurrent" tab.

3
Configure Recurrence Settings

In the Recurrent tab, configure:

  • Repeat Every: Set the interval (e.g., 1 Week, 2 Months)
  • Start From: Choose when the recurrence begins
  • Repeat On: For weekly tasks, select specific days (Mon, Tue, Wed...)
  • Until: Choose Forever, End Date, or Number of Repetitions
4
Preview Upcoming Tasks

The green preview box shows "Upcoming tasks will be created on:" with the next scheduled dates. Verify this matches your expectations.

5
Save the Task

Click Save. The task becomes a recurring template. New tasks will be automatically created by the scheduled cron job based on your settings.

Configure Scheduled Action (Cron Job)

How to Find

Go to Settings → Technical → Scheduled Actions and search for "Generate Recurring Project Tasks".

Configuration Options
  • Execute Every: Set frequency (e.g., 1 Day, 1 Hour)
  • Next Execution Date: When the cron will next run
  • Active: Toggle to enable/disable automatic generation
  • Number of Calls: -1 for unlimited executions
Tip

Click "Run Manually" to immediately generate all pending recurring tasks without waiting for the scheduled time.

Recurrent Tab in Action

Enable the "Recurring Task" checkbox to reveal the Recurrent tab with all configuration options.

How do I enable recurring tasks?

Open any project task, and check the "Recurring Task" checkbox. A new "Recurrent" tab will appear where you can configure the schedule.

Can I set tasks to repeat on specific weekdays?

Yes! When you select "Weeks" as the repeat unit, checkboxes for Mon-Sun appear, allowing you to pick exact days.

How are tasks generated?

A scheduled action (cron job) runs periodically and creates new tasks from your recurring templates when their next occurrence date is reached.

Is this compatible with Odoo Enterprise and Community?

Yes, this module works with both Odoo Enterprise and Community editions.
